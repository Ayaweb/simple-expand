simple-expand.js

simple jQuery plug-in to expand/colapse elements

You need: 

    
    <div id="#container">
        <a class="expander" href="#">click me</a>
        <div class="content">
            to hide this content.
        </div>
    </div>
  1. A container element on which you run the plugin.
  2. An element with class "expander"; Clicking this element expands/colapses the content.
  3. An element with class "content". This is the content to expand/colapse.

Recommended:

  • A css class like this to ensure all content looks collapsed while the document is being loaded. 
    .content {display:none;}
